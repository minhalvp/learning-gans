    PyTorch

    I have learnt that it is very important to understand each functions of the program and roughly understand the methods from PyTorch. This is because later, it will be much easier to understand your code and add comments which can be used as notes if you forget what a piece of code does. The PyTorch Library provides a neural network module which lets you create a neural network as a Class

    # The discriminator
class Discriminator(nn.Module):
    def __init__(self, in_features):
        super().__init__()
        self.disc = nn.Sequential(
            nn.Linear(in_features, 128),
            nn.LeakyReLU(0.01),
            nn.Linear(256, 1),
            nn.Sigmoid(),
        )

    def forward(self, x):
        return self.disc(x)

    The Discriminator inherits the PyTorch nn.module which gives the Discriminator class the neural network functionality. nn.Sequential allows us to create the neural network in order of layer ( nnLinear(in_features), 128 ) and an activation function ( nn.LeakyRELU(0.01) ). The outputs of a layer have to be the same dimensions as the input of the next layer as I learnt when I recieved this runtime error. The runtime error is complaining that it can not multiply the 2 matrices as the dimensions do not match

    
nn.Linear(in_features, 128),
nn.LeakyReLU(0.01),
nn.Linear(256, 1)
# 128 != 256

    The 256 indicates the number of inputs which is not the same as the number of outputs of the first Layer (128). PyTorch has built in optimizers for the networks. The role of the optimizers is to change the parameters of the model so that it yields better results.
